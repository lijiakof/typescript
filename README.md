# TypeScript 学习指南

## 安装 TypeScript
我们可以通过 npm 来安装 TypeScript：

```
npm install -g typescript
```

安装完成后我们就可以通过 `tsc` 命令来编译 .ts 文件了：

```
tsc hello.ts
node hello.js
```

如果你想通过一个命令来执行 .ts 文件，可以安装 ts-node 模块：

```
npm install ts-node --save-dev
```

现在就可以通过 `ts-node hello.ts` 来执行 .ts 命令了：

```
ts-node hello.ts
```
